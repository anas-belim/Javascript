JavaScript Concepts Explained

Learn JavaScript fundamentals with interactive examples

What is JavaScript?

JavaScript is a client-side scripting language used to make web pages dynamic, interactive, and user-friendly.

It runs inside the web browser without requiring any separate compiler.

// Simple JavaScript example
console.log("Hello, JavaScript!");

Variables & Data Types

Variables store data values. JavaScript has dynamic types.

let name = "John"; // String
const age = 30; // Number
var isStudent = false; // Boolean
let colors = ["red", "green"]; // Array
let person = {name: "John", age: 30}; // Object

Functions

Functions are blocks of code designed to perform a specific task.

function greet(name) {
  return "Hello, " + name;
}

// Arrow function (ES6)
const multiply = (a, b) => a * b;

Conditional Statements

Conditionals perform different actions based on different conditions.

let time = 14;

if (time < 12) {
  console.log("Good morning!");
} else if (time < 18) {
  console.log("Good afternoon!");
} else {
  console.log("Good evening!");
}

DOM Manipulation

The Document Object Model (DOM) represents the page structure as a tree of objects.

// Change text content
document.getElementById("demo").innerHTML = "Hello DOM!";

// Change style
document.getElementById("demo").style.color = "red";

This element will change.

Event Handling

JavaScript can respond to events like clicks, mouse movements, and keyboard input.

// Using addEventListener
document.getElementById("myBtn").addEventListener("click", function() {
  alert("Button clicked!");
});

Event result will appear here.
